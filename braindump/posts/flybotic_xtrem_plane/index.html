Flybotic Xtrem Plane

Fleeting

Avoid! The product uses an in-house battery. I cannot find any replacement and they don’t answer me when asking for directions.

It works well, but only one battery makes the enjoyment very short.

It looks like the parrot ones, except its dimensions (13 28 48) are nowhere to be found. It does not show any information on it, like a serial number that would help the search.

Opening it, I can see this component

along with this battery

Now we are getting somewhere. I can buy extra batteries, solder a JST connector on it and we should be good to go.

As a matter of fact, I can even use batteries from my hubsan x4 and 3D print a bigger enclosure.

Hubsan batteries dimensions are 39 x 18 x 8, while the xtrem plane is 752042 -> 42 x 20 x 7.5, therefore a little bit bigger. It might fit with an enclosure a little bit bigger. Note that soldering the JST connector will use a little bit more space.
