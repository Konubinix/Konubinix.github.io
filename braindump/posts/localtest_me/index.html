The entire domain name localtest.me—and all wildcard entries—point to 127.0.0.1 and ::1. So without any changes to your host file you can immediate start testing with a local URL.

— https://readme.localtest.me/

Examples:

http://localtest.me

http://newyork.localtest.me

http://mysite.localtest.me

http://redirecttest.localtest.me

http://sub1.sub2.sub3.localtest.me

— https://readme.localtest.me/
