Comment Calculer Le Taux/Coût D'un Congé Payé?

comment calculer le coût d’un congé payé

Il existe plusieurs méthodes.

compter le nombre de jours dans le mois

Soit diviser le brut de base par le nombre de jours de travail effectif ou assimilé sur le mois. Cela prend en compte les jours fériés. Ainsi, en juillet 2021, on compte 22 jours assimilés temps de travail effectif. Dans ce cas, les congés payés coûtent plus cher les petits mois

compter une moyenne grossière

Soit se baser sur une moyenne grossière égale à un mois de 21.67 jours.

21,67 = 5 jours dans la semaines x 52 semaines dans l'année / 12 nombre de mois dans l'année

Dans ce cas, un congé payé coûte M/21.67 mais est indémnisé (règle du dixième) M*12/10/25~=M/20.83 ou M est la paye mensuelle. Il rapporte donc M * (12/10/25-1/21.67) soit environ M*0.00185. Soit 1.85€ pour un salaire de 1K€.
