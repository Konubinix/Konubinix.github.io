weight to recharge

May be better to rely on the weight rather than on the color.

After repeated drying, the orange beads will lose indicating function by changing to other colors instead of dark green, but the humidity adsorbing function remains. Use the weighing method to confirm their saturation status when needed

— https://wisedry.com/user-guide/recharge-wisedry-silica-gel-sachets-medium-size-20-50-112-gram/ ([2025-05-26 Mon])

  • 20-gram silica gel packet weighs up to 26.4 grams
  • 50-gram silica gel packet weighs up to 66 grams
  • 112-gram silica gel packet weighs up to 148 grams

— https://wisedry.com/user-guide/recharge-wisedry-silica-gel-sachets-medium-size-20-50-112-gram/ ([2025-05-26 Mon])

time to recharge

20g x 5 packs (for reference): 700W – 3mins/side ( 3mins + 3mins ) 1000W – 2.5mins/side ( 2.5mins + 2.5mins )

50g x 2 packs (for reference): 700W: 50g x 2 packs – 2mins/side ( 2mins + 2mins ) 1000W: 50g x 2 packs – 2mins/side ( 2mins + 2mins )

50g x 4 packs (for reference) 700W: 50g x 4 packs – 4mins/side ( 4mins + 4mins ) 1000W: 50g x 4 packs – 3.5mins/side ( 3.5mins + 3.5mins )

112g x 1 pack (for reference) 700W: 112g x 1 pack – 3.5mins/side ( 3.5mins + 3.5mins ) 1000W: 112g x 1 pack – 3.5mins/side ( 3.5mins + 3.5mins )

112g x 2 packs (for reference) 700W: 112g x 2 packs – 5.5mins/side ( 5.5mins + 5.5mins ) 1000W: 112g x 2 packs – 4.5mins/side ( 4.5mins + 4.5mins )

— https://wisedry.com/user-guide/recharge-wisedry-silica-gel-sachets-medium-size-20-50-112-gram/ ([2025-05-26 Mon])

If the above suggested time fails to have them fully recharged, add 30s-60s until the green beads fully restore to orange

— https://wisedry.com/user-guide/recharge-wisedry-silica-gel-sachets-medium-size-20-50-112-gram/ ([2025-05-26 Mon])
