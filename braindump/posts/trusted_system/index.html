Trusted System

Fleeting

A system in which you can capture stuffs (ideas, actions, projects, engagements) so that your brain can stop thinking about those.

progressive refinement

When capturing stuff, you are likely primed in some direction. The fact you have clear ideas about what you want to capture bias the location you store the stuff (wysiati).

When you want to retrieve the stuff, your brain is not primed the same way, and the path to get to that data is not the same.

The network aspect makes it feasible to retrieve the data, although needing some effort. In that situation, it is important to adjust the system to make the information retrieval easier for your future self.

A bit like backpropagation, you adjust the network so that it will respond better next time.
