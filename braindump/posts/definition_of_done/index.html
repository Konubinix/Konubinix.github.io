You have to try to envision what it feels like to have it done.

In gtd, visualize what done means so that you put yourself in the state of mind of doing toward this state of done. Supposedly your reticular system will make easier to grasp things that will help you in this goal. It is like when riding a bike, when you look at the left, the bike “magically” goes to the left. There are supposedly plenty of unconscious mechanisms that will make you go into the direction of done, provided you visualized it.

Definition of Done is a formal description of the state of the Increment when it meets the quality measures required for the product.

— https://scrumguides.org/scrum-guide.html
