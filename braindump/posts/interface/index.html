Interface (informatique) — Wikipédia

Une interface définit la frontière de communication entre deux entités

— https://fr.wikipedia.org/wiki/Interface_(informatique)

réfère généralement à une image abstraite qu’une entité fournit d’elle-même à l’extérieur

— https://fr.wikipedia.org/wiki/Interface_(informatique)

distinguer les méthodes de communication avec l’extérieur et les opérations internes,

— https://fr.wikipedia.org/wiki/Interface_(informatique)

autorise à modifier les opérations internes sans affecter la façon dont les entités externes interagissent avec elle

— https://fr.wikipedia.org/wiki/Interface_(informatique)

Variétés :

Une interface humain-machine permet des échanges entre un humain et une machine ; Une interface de programmation permet des échanges entre plusieurs logiciels ; L’interface qui est présentée à l’utilisateur est nommée interface utilisateur ; Les interfaces entre des composants de matériel informatique sont des interfaces physiques.

— https://fr.wikipedia.org/wiki/Interface_(informatique)
