Adresse De La Machine Hôte Dans Docker

Fleeting

adresse de la machine hôte dans docker, quand on utilise le network bridge (localhost)

docker inspect bridge | jq -r .[0].IPAM.Config[0].Gateway

172.17.0.1

Attention, si on fait du docker-in-docker, ce gateway peut être différent

docker crée des sous-réseaux pour discuter avec les conteneurs. Les conteneurs docker peuvent interagir avec l’hôte via cette adresse. C’est le 127.0.0.1 dans le conteneur.
