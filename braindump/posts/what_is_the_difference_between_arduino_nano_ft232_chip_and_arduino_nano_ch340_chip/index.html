What is the difference between arduino nano (FT232 chip) and arduino nano CH340 chip ?

FT232 and the CH340 are both serial to usb ICs. The CH340 is often used on Chinese Arduino clones

— https://forum.mysensors.org/topic/3508/what-is-the-difference-between-arduino-nano-ft232-chip-and-arduino-nano-ch340-chip

no difference on the user side, expect that you need to download and install drivers on windows for the CH340. (On Linux included)

— https://forum.mysensors.org/topic/3508/what-is-the-difference-between-arduino-nano-ft232-chip-and-arduino-nano-ch340-chip

If you use a controller running on RPI or some other modern OS it usually recognises the ch340 modules just fine.
