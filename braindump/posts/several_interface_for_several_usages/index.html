Several Interface for Several Usages

Fleeting

I use in my daily life tools that are CLI, TUI, GUI and NUI, depending on the usage.

Often, people think I am a “geek” to “prefer CLI tools instead of GUI tools”. I admit I may be a bit geeky, but in my humble opinion, the reason is not at all that supposedly prefer CLI tools. It is more that I use the appropriate tool for the job to be as fast as possible.

I tend to remind those people that

  • my MUA is a NUI tool,
  • my life hacking editor is a GUI tool,
  • my 3D modeling tool is a GUI tool,
  • my 2D modeling tool is a GUI tool,
  • my web browsing tool is a GUI tool,
  • my pomodoro is a NUI tool,

I decided to believe that those people are subject to a cognitive dissonance of the form:

  • we use a GUI tool to do X
  • we are clever
  • hence clever people use a GUI tool to do X (association)
  • Sam uses a CLI (or TUI) tool to do X
  • Sam appears to be clever

Then, they minimize this cognitive dissonance by minimizing my status, putting me in the “geek” category.
