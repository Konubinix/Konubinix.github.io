Quickly Printing a New Hook

Fleeting

tag: learning 3D printing

With cadquery,

import cadquery as cq


path = (
    cq.Workplane("XY")
    .threePointArc(
        (4, 4),
        (8, 0),
    )
    .lineTo(8, -3)
    .threePointArc(
        (4, -7),
        (0, -3),
    )
)
profile = (
    cq.Workplane("XZ")
    .circle(0.5)
)

result = (
    profile.sweep(path)
)

result = (
    result.faces("|Y").fillet(0.2)
)

cq.exporters.export(
    (
        result.val()
        .scale(5) # get back to the approriate scale
    ),
    out,
)
return out

()

Printed with 100% infill.

Without a brim, the model warped a little bit, but it was still usable.
