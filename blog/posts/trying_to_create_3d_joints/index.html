Trying to Create 3D Joints

tag: learning 3D printing

I want to have 3D joints ready to be used in my project.

I don’t need nor do I have the skill to create sophisticated joints like cantilever snap-fit yet, so I’ll start with the basics.

Dovetail joint

When I need to assemble to plane parts, I can try using such kind of joint.

I don’t know though how thin I can make the part.

Let’s try with 2mm large parts in and a tail of 2cm.

I’m particularly worried about the needed clearance.

Any joint needs some space between parts in order for them to fit together. This space is called clearance. 0.3 mm is good for most 3D printers, though this can fluctuate between 0.2 and 0.6 mm

— https://all3dp.com/2/3d-printed-joints-the-basics/
