Printing a Meeple of a Kid Goat and Discovering 2D -> 3D Modeling

tag: learning 3D printing

I needed to create a new meeple to replace the one we lost.

First, take a picture of the meeple.

Then, using gimp, get the black and white bitmap of the surface to use.

a.png

Then, using inkscape, you can convert that image to a svg path.

a.svg

Then, simply import this svg into blender.

Scale the model so that it matches the original one. In our case, the meeple appears to have:

  • a width of 2cm
  • and a height of 0.5cm

You must convert the curve to a mesh so that it will be extruded without errors.

Finally, you get the following blender file:

untitled.blend

And you can convert it to the following stl file, ready to be put in cura to

()

Now, our kid goats family has a new member.
