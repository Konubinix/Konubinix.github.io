untitled.blend

()

strengthen it again

Well, after a few weeks, it broke again…

To make the cylinder more robust, I can try :

  • printing it vertically,
  • printing it slower,
  • printing it with hotter nozzle (to make the layers stick better),
  • printing it using petg,
  • covering it with epoxy resin,

Let’s try first epoxy resin, since it let me reuse the current print.

The resin took a long time to harden, so I finally hang the object and retrieved the drops in a spoon that I poured again on the object, until it was hard enough not to flow.

Now, let’s wait and see how strong it has become.

So far, it looks pretty strong. Have known this, it would have avoided printing several times in this project.
